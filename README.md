# Full-Stack-Task-2
Kanban Board
A modern, responsive Kanban Board web application built using HTML, CSS, and JavaScript.  
This project helps organize tasks efficiently using a simple To Do to Doing to Done workflow.
Features:-
- Add new tasks  
- Edit tasks directly  
- Delete tasks  
- Drag and drop tasks between columns  
- Automatic task count per column  
- Fully responsive for desktop and mobile  
- Built with pure HTML, CSS, and JavaScript without libraries
